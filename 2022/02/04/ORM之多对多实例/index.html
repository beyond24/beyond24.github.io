    ORM之多对多实例

    案例分析：

    一个用户可以选多门课程；

    一个课程也可以被多个用户选择

    model.py设计

    class UserInfo(models.Model):
    name = models.CharField('姓名' , max_length=32)
    age = models.IntegerField('年龄')



class Course(models.Model):
    name = models.CharField('课程名称',max_length=32)
    user = models.ManyToManyField(
        to='UserInfo',
    )

    view.py

    def testManyToMany(request):

    # 前两个用户添加体育课
    user_list = UserInfo.objects.filter(id__in=[1,2])
    course_obj = Course.objects.filter(name='体育').first()
    print(course_obj)

    course_obj.user.add(*user_list)

    #查选体育课的所有用户queryset，显示用户名
    u = course_obj.user.all().values('name')
    print(u)

    #反向查询wang选的所有课程queryset，显示课程名
    wang_obj = UserInfo.objects.filter(name='wang').first()
    c = wang_obj.course_set.all().values('name')
    print(c)

    
    return HttpResponse('成功')

    控制台输出

    • 添加

      • 除了连接两表的第三张表，其余添加操作与单表插入一样
      • 第三张表添加：
        • (写有连接字段的)对象.表名.add()

    • 正向查询

      • 从写有连接的表查询另一表达的操作
      • 使用对象+表名

    • 反向查询

      • 从未设连接的表查
      • 使用对象+(连接表名)__set

    补充：

    之前我们每次查询都需要去走路由，在视图函数里面去写对应的代码

    import os

if __name__ == '__main__':
    # 加载Django项目的配置信息
    os.environ.setdefault("DJANGO_SETTINGS_MODULE", "Django_test.settings")
    # 导入Django，并启动Django项目
    import django

    django.setup()

    from app01.models import *
    
    ## 这里写你的查询语句
        # 正向查询
    course_obj  = Course.objects.get(name='体育')
    # 等同于
    # course_obj  = Course.objects.filter(name='体育').first()
    print(f'选修体育课的人数是:{course_obj.user.count()}')

    # 反向查询

    输出结果：

    选修体育课的人数是:2
    <QuerySet [{‘name’: ‘语文’}, {‘name’: ‘体育’}]>

    基于双下划线进行链接查询和基于对象连接查询

    • 基于双下划线的方法是可以在过滤和取值时直接进行连表操作
       # 查询选修了体育课的用户信息queryset
    user_list = UserInfo.objects.filter(course__name='体育')
    print(user_list.values('name'))
    # 查询用户名为wang选修的课程queryset
    course_list = Course.objects.filter(user__name='wang')
    print(course_list.values('name'))

    输出结果：

    <QuerySet [{‘name’: ‘wang’}, {‘name’: ‘gp’}]>
    <QuerySet [{‘name’: ‘体育’}, {‘name’: ‘语文’}]>

    • 使用对象查询实现相同的效果
    user_obj2 = UserInfo.objects.get(name='wang')
user_list2 = user_obj2.course_set.all().values('name')
print(user_list2)

course_obj2 = Course.objects.get(name='体育')
course_list2 = course_obj2.user.all().values('name')
print(course_list2)

    输出结果：

    <QuerySet [{‘name’: ‘语文’}, {‘name’: ‘体育’}]>
    <QuerySet [{‘name’: ‘wang’}, {‘name’: ‘gp’}]>
